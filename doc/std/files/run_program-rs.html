program_res

A resource that manages the destruction of a program object

program_res ensures that the destroy method is called on a program object in order to close open file descriptors.

start_program

fn start_program(prog: str,
args: [str]) -> @program_res

Spawns a process and returns a boxed program_res

The returned value is a boxed resource containing a program object that can be used for sending and recieving data over the standard file descriptors.  The resource will ensure that file descriptors are closed properly.

Parameters

progThe path to an executable
argsVector of arguments to pass to the child process
