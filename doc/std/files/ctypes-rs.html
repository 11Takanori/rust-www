type intptr_t = uint

A signed integer with the same size as a pointer.  This is guaranteed to always be the same type as a Rust `int`

uintptr_t

type uintptr_t = uint

An unsigned integer with the same size as a pointer.  This is guaranteed to always be the same type as a Rust `uint`.

void

type void = int

A type, a pointer to which can be used as C `void *`

Note that this does not directly correspond to the C `void` type, which is an incomplete type.  Using pointers to this type when interoperating with C void pointers can help in documentation.
