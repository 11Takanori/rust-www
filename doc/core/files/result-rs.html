chain

fn chain<T, copy U, copy V>(res: t<T, V>,
op: block(T) -> t<U, V>) -> t<U, V>

Call a function based on a previous result

If `res` is <ok> then the value is extracted and passed to `op` whereupon `op`s result is returned. if `res` is <err> then it is immediately returned.  This function can be used to compose the results of two functions.

Example

let res = chain(read_file(file), { |buf|
  ok(parse_buf(buf))
})
