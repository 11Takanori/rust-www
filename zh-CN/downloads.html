这里是一些早期版本的 存档。

〔注〕 在 Windows 操作系统中有两个可供选择 ABIs 的版本。使用已经安装的 (MSVC) ABI 需要 Visual Studio, ，而使用 GNU ABI 则需要 GCC 工具链. Which version of Rust you need depends largely on what C/C++ libraries you want to interoperate with: for interop with software produced by Visual Studio use the MSVC build of Rust; for interop with GNU software built using the MinGW/MSYS2 工具链 use the GNU build.

MSVC builds of Rust additionally require the Microsoft Visual C++ build tools for Visual Studio 2013 或更高版本。 The easiest way to acquire the build tools is by installing Microsoft Visual C++ Build Tools 2015 which provides just the Visual C++ build tools. Alternately, you can 安装 Visual Studio 2015 or Visual Studio 2013 and during install select the "C++ tools". No additional software installation is necessary for basic use of the GNU build.

Rust 对 GNU ABI 的支持更加成熟，建议作为典型用途。
